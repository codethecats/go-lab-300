    ถ้า เท่ากับ -> พิมพ์ "แนะนำให้สั่งคู่กับต้มจืดไข่น้ำ"

iii. ตรวจสอบ order == "ไข่เจียว" ไหม ?
    ถ้า เท่ากับ -> พิมพ์ "แนะนำให้สั่งคู่กับไข่ลูกเขย"
    แต่ถ้า ไม่เท่ากับ

    แต่ถ้า ไม่เท่ากับ -> พิมพ์ "ร้านเรามี ไข่ดาว, ไข่เจียว, ไข่ต้ม"
